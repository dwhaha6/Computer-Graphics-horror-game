# 리포트 첨부 이미지 폴더

여기에 **게임 캡처**를 넣고, `REPORT.md`의 `〔캡처: …〕` 자리에 아래처럼 끼워 넣으면 됩니다.

```markdown
![설명 캡션](images/파일이름.png)
```

> 캡처 팁: 게임 중 **`F11`(전체화면)** 후 캡처하면 깔끔합니다. DDGI 비교는 `?debug` / GI off는 `?nogi` URL 파라미터를 붙여 같은 구도로 두 장 찍으세요.
