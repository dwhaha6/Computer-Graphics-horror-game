## 2. 기획 (채점: 기획 20)

### 2.1 컨셉 · 레퍼런스
> ✍️ 직접 작성 — RE7에서 차용한 요소(감금·곤충·부패한 가정집·"가족"이라 부르는 가해자)와 본 게임만의 차별점(할머니 납치범, 흔적 복원 메커니즘, 옆방 동료 김씨, 멀티 엔딩).

〔캡처: 방 전경 (식탁·의자·가구 와이드 샷)〕

### 2.2 핵심 게임플레이 루프
```
① 결박 해제   식탁의 숟가락(E) 줍기 → F로 수갑 풀기
② 자명종 퍼즐  시계를 반시계로 돌려 분침(=열쇠) 획득   ※ 시계방향=자명종→할머니(40초)
③ 장롱        분침으로 장롱 열기 → 쪽지(암호 단서)     ※ 최초 개방=할머니(30초)
④ 서랍 퍼즐    쪽지 단서로 4자리(9219) → 서랍 개방 → 사진 2장 (+벌레 연출)
⑤ 그림 퍼즐    사진의 토글 규칙 → 영단어(GOLD) → 좌측 암호장치 → 사다리  ※ 정답=할머니(20초)
⑥ 탈출/엔딩    옆방 김씨와 거래(드라이버) → 환풍구로 탈출, 선택에 따라 결말 분기
```
> ✍️ 직접 작성 — 각 단계의 순서 근거, 긴장과 해소의 리듬.

〔캡처: 분침 획득〕 〔캡처: 서랍 4자리 자물쇠 UI〕 〔캡처: 그림 퍼즐(사진 인스펙트)〕

### 2.3 납치범(할머니) 복귀 메커니즘 — 게임의 심장
- **트리거**: 자명종 오답(40s) · 장롱 최초 개방(30s) · GOLD 정답(20s) · 칼 소리치기(8s) · 둠런 중앙선(15s)
- **점검 6종**(흔적 복원): 장롱 닫힘 · 손전등 제자리 · 서랍 닫힘 · **사다리 회수** · 의자 착석 · 수갑 잠김
- **행동 분기**: 들어와 두리번 → (이상 없음) 방을 둘러보고 **그냥 나감** / (이상 있음·은신) **코앞까지 다가와 점프스케어**
- **은신**: 장롱 안에 숨기(F) — 들키면 더 끔찍한 연출
> ✍️ 직접 작성 — "흔적을 원상복구하고 다시 묶인 척"이라는 규칙이 만드는 심리적 압박.

〔캡처: 할머니 입장(문틈/실루엣)〕 〔캡처: 점프스케어〕 〔캡처: 카운트다운 타이머 + "할머니 올 때" 자막〕

### 2.4 긴장 유발 장치 (상시 공포)
- 벌레 스프(숟가락으로 먹기, 3연속=중독사) · 서랍 개방 시 바퀴벌레 연출 · 랜덤 벽 긁힘/삐걱
- 쇠창살 위 **아기 사진**(그쪽을 보면 비명) · 어두운 방(랜턴 미소지 시)
> ✍️ 직접 작성

〔캡처: 벌레 스프 그릇〕 〔캡처: 아기 사진〕 〔캡처: 구더기 사망 이미지〕

### 2.5 멀티 엔딩 (3종) + 분기 설계
| 엔딩 | 조건 | 결말 |
|---|---|---|
| **Hidden** | 의자 밑 **칼** 선택 → 소리쳐 부르고 장롱에서 급습 | 살인 → 신문기사 |
| **Normal** | 옆방에 **분침(가짜 열쇠)** 건넴 → 환풍구 탈출 | 혼자 탈출하나 할머니가 따라옴 (둘 다 죽음) |
| **True** | 의자 밑 **열쇠** → 옆방에 건넴 → 함께 탈출 | 둘 다 생존·자수 |
- **히든 분기 진입 조건**: 사다리 획득 **이후** 벌레 스프 섭취 + 어두운 방 → 벽에 푸른 글씨(의/자/밑) → 의자 밑에서 칼/열쇠 택1(나머지는 지하로 소실)
> ✍️ 직접 작성 — 선택의 도덕적 무게, 각 엔딩의 의미.

〔캡처: 의자 밑 칼/열쇠 선택 UI〕 〔캡처: Hidden Ending〕 〔캡처: Normal Ending〕 〔캡처: True Ending〕

### 2.6 죽음 · 세이브포인트 설계
- **1번째 죽음**: 리셋 X → 구간 체크포인트로 부활(아이템·진행 유지, 위험 트리거 초기화)

> ✍️ 직접 작성 — 재도전 마찰을 줄이면서 공포를 유지한 의도.

## 4. GI 기술 — DDGI (채점: GI 20) ★ 핵심

### 4.1 DDGI 개념 요약
> ✍️ 직접 작성 — DDGI가 무엇인지(프로브 격자에 방향별 irradiance 저장 → 실시간 확산 GI), 왜 이 게임에 적합한지(움직이는 광원=랜턴/할머니의 차가운 빛 → 간접광이 실시간으로 출렁이며 공포 분위기 강화).

〔캡처: 랜턴을 들고 이동할 때 벽에 번지는 간접광 (GI on)〕

### 4.2 파이프라인 (직접 구현 부분 강조)
```
① 프로브 격자 배치   방 공간에 3D 그리드로 probe 배치
② 프로브 갱신        각 probe에서 저폴리 프록시 씬을 큐브/레이로 샘플
③ 옥타헤드럴 인코딩   방향별 irradiance + depth(거리·거리²)를 옥타맵 아틀라스에 저장
④ 셰이딩 적용        머티리얼에서 표면 노멀로 옥타맵 조회 + 보간
⑤ Chebyshev 가시성   depth 통계로 빛샘(light leak) 억제
⑥ 분할 갱신(amortized) 매 프레임 일부 프로브만 갱신 → 실시간 성능
```
> ✍️ 직접 작성 — 위 각 단계에서 본인이 구현/조정한 부분, 핵심 셰이더 수식(옥타 인코딩·Chebyshev), `onBeforeCompile` 머티리얼 주입 등. (코드: `src/gi/ddgi/*`)

〔캡처: `?debug` 진단 뷰〕 〔코드: ProbeVolume / 옥타 인코딩 핵심 스니펫〕

### 4.3 GI가 "게임 메커니즘"이 되는 지점
> ✍️ 직접 작성 — 어두운 방에서 랜턴으로 비춰야 단서가 보임, 할머니의 차가운 광원이 들어올 때 간접광이 바뀜 = 공포 큐이자 GI 쇼케이스. (GI On/Off 비교 캡처 권장)

## 7. 문제와 해결 (개발 로그)
| 문제 | 원인 | 해결 |
|---|---|---|
| 할머니가 책상을 통과 | 경로상 가구 미충돌 | 축별 슬라이드 충돌 회피(footprint) |
| 시선이 뒷벽으로 튐 | 비카메라 lookAt 핸들니스 | 더미를 PerspectiveCamera로 |
| 대화 후 게임 멈춤 | 섹션 대화 누락 → 예외 → rAF 루프 사망 | 폴백 + tick try/catch |
| 엔딩 이미지 즉시 사라짐 | 클릭 버블링으로 즉시 reload | 핸들러를 버튼에 고정 |
| 비명 사운드가 빵빠레처럼 | 순수 사인 배음 | 노이즈 공명 기반으로 재합성 |
> ✍️ 직접 작성 — 그 외 본인이 겪은 문제 추가.
